For the function above, I am starting with no predictor variables. The function then take the AIC for every model that contains one predictor and choose the lowest AIC. The function continues adding predictors until adding any predictor would result in a higher AIC than if it were to not add any more predictors.

The formula to determine the AIC is as follows:

\(n \log\left(\frac{{\text{{RSSM}}_k}}{{n}}\right) + 2pM\)
